# DevDay 2023 - Blazor beyond the web

## Description

La technologie Blazor est maintenant bien connue, que ce soit via un Serveur ou en WebAssembly.
Sa simplicité et son approche ont maintenant séduit de nombreux développeurs.

Dans cette session nous allons aborder ce que l'on appelle Blazor Hybrid.

Imaginez-vous utiliser un même composant dans tous vos projets: 
applications existantes WPF ou Windows Forms.
Imaginez que ce même composant se retrouve dans votre application MAUI. 
Cela vous permet de migrer progressivement un logiciel.

Mais pourquoi ne pas aller plus loin ?
Imaginez que vous utilisez ce même composant Blazor dans votre projet Angular ou React. 
Ou même dans votre site PHP ?

Et bien c'est ce que nous allons vous montrer !
